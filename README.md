# ✈️ AI Trip Planner

An AI-powered travel planning web application built using **Django** and **Google Gemini AI**.

## 🚀 Features

- 🔐 User Registration & Login
- 🤖 AI-generated travel itinerary using Gemini AI
- 🌦 Live Weather Information (OpenWeather API)
- 🖼 Destination Images (Pexels API)
- 💰 Budget Analytics
- 📍 Google Maps Integration
- 📄 Download Trip Plan as PDF
- 🗑 Save and Delete Trips

## 🛠 Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite3
- Google Gemini API
- OpenWeather API
- Pexels API
- ReportLab

## ⚙ Installation

```bash
git clone https://github.com/vinayakrkalburgui123/AI_Trip_Planner.git

cd AI_Trip_Planner

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
